# 环岛路骑行指南

最美海岸线，吹着海风看日落。环岛路不仅有无敌海景，还有丰富的休闲娱乐设施。

## 🏖️ 必经景点
- **白城沙滩**：厦大南门附近，适合漫步。
- **书法广场**：极具文化底蕴的休息点。
- **音乐广场**：常有流浪歌手，氛围感十足。
